Introduction¶

This documentation describes the Python3 resources for implementing decision aid algorithms in the context of a bipolarly-valued outranking approach ([1], [2]). These computing resources are useful in the field of Algorithmic Decision Theory and more specifically in outranking based Multiple Criteria Decision Aid (MCDA).
