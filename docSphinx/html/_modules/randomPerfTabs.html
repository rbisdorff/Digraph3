        * integerWeights := Boolean (True = default)

            | 'pref':(1,0),
            | 'veto':(numberOfActions,0)
            | } (default)

                 weightDistribution = 'equisignificant', weightScale=None,\

        """

            sumWeights = sum(weightsList)
        else:
